# faimatantemplate


## Описание

Это проект faimatantemplate (Faculty of Artificial Intelligence; MAThematical ANalysis; TEMPLATE), представляющий **шаблон LaTeX-проекта** для создания учебных материалов по математике (и смежным дисциплинам), разработанный в рамках работы над учебником по математическому анализу для факультета искусственного интеллекта МГУ.

> Шаблон распространяется свободно (при условии указания автора дизайна в тексте и/или ссылки на данный репозиторий): вы можете использовать его как отправную точку для собственных конспектов, методичек, курсовых, дипломных работ и книг.


## Основные возможности

- **Семантические цветные блоки.**

Готовые команды-обёртки для типовых математических элементов с единой сквозной нумерацией и собственным цветовым кодом:
- `\defi` — определения (**О**),
- `\stat` — утверждения (**У**),
- `\lemm` — леммы (**Л**),
- `\theo` — теоремы (**Т**),
- `\conc` — следствия (**С**),
- `\exam` — примеры (**П**),
- `\note` — замечания (**З**),
- `\algo` — алгоритмы (**А**),
- `\code` — программный код (**К**),
- `\task` — задачи.

Каждый блок поддерживает звёздную (ненумерованную) форму, опциональные аргументы (доказательство/решение, примеры, замечания) и автоматическую вставку знака завершения доказательства.

- **Единая система обозначений.**

Файл `helpers.tex` содержит набор готовых математических команд и текстовых сокращений (числовые множества, окрестности, пределы, производные, кванторы, операции над множествами и~т.д.), обеспечивающих единообразие записи во всём документе.

- **Авторское оформление заголовков.**
  
Стилизованные заголовки глав, разделов и подразделов на основе TikZ (шестиугольные и прямоугольные «бейджи» с градиентами и бликами, декоративные вертикальные метки для ненумерованных заголовков).

- **Вставка программного кода с выводом.**
  
Команда `\code` позволяет подключать внешние файлы с исходным кодом на Python (через `minted`) и при необходимости автоматически печатать вывод программы в том же блоке. Lua-препроцессор (`code_filter.lua`) поддерживает специальные бейджи `\cmo{N}` для связывания строк кода со строками вывода.

- **Псевдокод и формулы.**

Подключён и русифицирован пакет `algpseudocode` для записи алгоритмов; настроена аккуратная типографика выключных формул, переносов, систем и кусочных определений.

- **Готовая типографика книги.**

Настроены геометрия страницы, шрифты (STIX Two Text / Math, Libertinus Sans, CMU Typewriter), колонтитулы, микротипографика, гиперссылки и закладки PDF, поддержка `todonotes` для рабочих пометок (`\td`, `\fx`).

- **Приложение «Руководство по стилю оформления».**

В шаблон включено отдельное объёмное приложение (`content/appendix/styleguide/`), где **систематически зафиксированы все стилистические соглашения** и приведены таблицы со всеми используемыми командами (как авторскими, так и стандартными LaTeX).

Это приложение выполняет двойную роль:
- служит **справочной памяткой** по математическому набору и единому стилю для вас и ваших соавторов;
- может использоваться как **промпт для больших языковых моделей**: если передать ИИ-ассистенту эти правила вместе с исходным LaTeX-кодом, он сможет автоматически приводить документы к единообразному и аккуратно структурированному виду.


## Структура проекта

```
.
├── book.tex                  # Главный файл (содержит полное оглавление)
├── code/                     # Файлы с программным кодом для вставки
├── content/
│   ├── cover.tex             # Обложка
│   ├── cover_verso.tex       # Оборот обложки (выходные данные)
│   ├── intro.tex             # Введение
│   ├── conclusion.tex        # Послесловие
│   ├── l-01/ ...             # Материал по лекциям (главам)
│   └── appendix/styleguide/  # Руководство по стилю оформления
└── images/                   # Рисунки и логотипы
├── style/
│   ├── color.tex             # Цветовые схемы (базовые и семантические цвета)
│   ├── base.tex              # Пакеты и базовые настройки документа
│   ├── helpers.tex           # Авторские команды и сокращения
│   ├── block.tex             # Окружения для определений, теорем и т.п.
│   ├── base_chapter.tex      # Оформление заголовков глав
│   ├── base_section.tex      # Оформление заголовков разделов
│   ├── base_subsection.tex   # Оформление заголовков подразделов
│   └── code_filter.lua       # Lua-препроцессор для блоков \code
```


## Требования

- Движок **LuaLaTeX** (используется `fontspec`, `unicode-math`, прямые вызовы Lua).
- **Шрифты:** STIX Two Text, STIX Two Math, Libertinus Sans, CMU Typewriter Text, DejaVu Sans Mono.
- **Pygments** (Python-пакет) для работы пакета `minted` (требуется флаг `-shell-escape`).
- Полный дистрибутив TeX (рекомендуется TeX Live актуальной версии).


## Сборка LaTeX-документа

В репозитории содержится python скрипт `build.py`, при запуске которого будет произведена автоматическая сборка документа.

Альтернативно можно использовать следующую команду (стоит выполнить дважды для корректной сборки):
```bash
lualatex -synctex=1 -interaction=nonstopmode -file-line-error -halt-on-error -shell-escape main.tex
```
